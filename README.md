# Noble Scarlet 字体备份库

## 🔍 字体简介
Noble Scarlet 是微软为替代经典「微软雅黑」开发的新一代屏显黑体项目早期版本（内部代号），由方正字库基于 **兰亭黑 Pro** 进行二次开发，西文部分整合 Segoe UI 字体资源。该字体原本计划成为 Windows 系统默认字体，优化了高分屏显示体验和多语言兼容性，但最终因项目调整被微软雪藏。

作为一款专为现代显示设备设计的黑体字体，其核心优势在于：

- 收敛式大中宫设计，提升小字号易读性

- 优化的 hinting 机制，减少锯齿和模糊问题

- 完整覆盖多字符集，支持中英文、西里尔字母、希腊字母及特殊符号

- 适配 4K / 高分屏显示，视觉效果接近 macOS 字体细腻度

## ⚠️ 注意事项

1. 本字体为未正式发布的测试版本，可能存在少量兼容性问题
1. 本仓库仅提供字体备份存档，所有版权归属原开发方（微软 / 方正字库），请勿用于商业用途
1. 替换系统默认字体前请务必备份原版文件，避免系统异常
