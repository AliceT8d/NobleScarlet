# Noble Scarlet 字体备份

## 🔍 字体简介
网传 Noble Scarlet 是微软为替代经典「微软雅黑」开发的新一代屏显黑体项目早期版本，该字体泄露于Windows 10 预览版，英文原文请参见 [Announcing Windows 10 Insider Preview Build 17025 for PC](https://blogs.windows.com/windows-insider/2017/10/25/announcing-windows-10-insider-preview-build-17025-pc/)，关于字体更新的信息中文译文如下：

> 微软雅黑字体升级说明：中国的预览体验成员想必都知道，微软雅黑是我们在简体中文系统中用于显示 Windows 界面文字的字体。我们已听取大家的反馈，并在本次版本中对该字体进行更新，以提升其易读性、对称性与整体显示效果。
>
> 本次更新的重点优化内容如下：
>
> - 为微软雅黑新增三款字重：Semilight、Semibold、Heavy。该新增字重将在简体中文版本中默认安装，非简体中文版本可通过可选功能下载获取。
> - 对字形进行比例调整，优化字符间距，提升阅读舒适度与字符辨识度。
> - 采用全新的字体微调算法对字体进行调校，有效改善小字号下字形的锯齿问题。
> - 结合用户反馈，重新设计了新版字体中的所有标点符号与特殊符号。
>
